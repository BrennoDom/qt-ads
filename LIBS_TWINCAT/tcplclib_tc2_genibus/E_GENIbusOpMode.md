# E\_GENIbusOpMode

Adjustable control modes.

```
TYPE E_GENIbusOpMode :  
(  
  eGENIbusOpModeUnknown := 0,  
  eGENIbusOpModeStop    := 1,  
  eGENIbusOpModeStart   := 2,  
  eGENIbusOpModeMin     := 3,  
  eGENIbusOpModeMax     := 4  
);  
END_TYPE
```
