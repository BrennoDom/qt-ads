# E\_GENIbusOS

OS (Operation-Specifier) in the command telegram.

```
TYPE E_GENIbusOS :  
(  
  eGENIbusGET  := 0,  
  eGENIbusSET  := 1,  
  eGENIbusINFO := 2  
);  
END_TYPE
```
