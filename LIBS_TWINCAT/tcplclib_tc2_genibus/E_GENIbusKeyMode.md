# E_GENIbusKeyMode

## Library
tcplclib_tc2_genibus

## Category
Motion Control

# E\_GENIbusKeyMode

Disablement of the parameterization option on the GENIbus device.

```
TYPE E_GENIbusKeyMode :  
(  
  eGENIbusKeyModeLocked   := 0,  
  eGENIbusKeyModeUnlocked := 1  
);  
END_TYPE
```
