# Tc2\_NciXFC

The Tc2\_NciXFC library facilitates the precisely timed recording of relative path distances and the path-precise output of digital signals in connection with the EtherCAT XFC terminals. The required output function blocks are included in the Tc2\_MC2\_XFC library.

A precise correlation between time and relative path position can thus be established at any time.

The library provides various function blocks for the calculation of the timestamps or positions.
