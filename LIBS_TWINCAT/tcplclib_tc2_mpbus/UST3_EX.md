# UST3\_EX

Voltage scaling

```
TYPE UST3_EX :  
(  
  Ex_1mV   := 0,  
  Ex_250uV := 1  
)  
END_TYPE
```

**Ex\_1mV:** Scaling 0..11 V

**Ex\_250uV:** Scaling 0..3 V
