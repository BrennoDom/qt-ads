# Introduction

The library Tc2\_SUPS contains functions and function blocks required for controlling the 1-second UPS (SUPS).

## Function blocks

| Name | Description |
| --- | --- |
| [FB\_S\_UPS](ms-xhelp:///?Id=beckhoff-1ce5-4c5b-8c7d-2e323f4ae1e8) | Function block for controlling the 1-second UPS of CX50x0 Embedded PCs |
| [FB\_S\_UPS\_CB3011](ms-xhelp:///?Id=beckhoff-9edb-4894-95f4-da724bf46e46) | Function block for controlling the 1-second UPS of PCs with CB3011 board |
| [FB\_S\_UPS\_CX51x0](ms-xhelp:///?Id=beckhoff-bda3-48d4-8b63-cadc4f700fa2) | Function block for controlling the 1-second UPS of CX51x0 Embedded PCs |
| [FB\_S\_UPS\_CX9020\_U900](ms-xhelp:///?Id=beckhoff-c47f-4fd6-be39-a295066fc1a2) | Function block for controlling the 1-second UPS of CX9020-U900 Embedded PCs |
| [FB\_S\_UPS\_BAPI](ms-xhelp:///?Id=beckhoff-88ac-4eba-bf71-61428646c378) | Function block for controlling the 1-second UPS of devices with BIOS-API from version v1.15 |
| [FB\_NT\_QuickShutdown](ms-xhelp:///?Id=beckhoff-b997-40bb-8b62-3e706cbb38c0) | Internal function block for quick shutdown, which is used by the FB\_S\_UPS. |
