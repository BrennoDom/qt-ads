# Addressing

## Function blocks

| Name | Description |
| --- | --- |
| [FB\_DALIV2AddressingIntRandomAddressing](ms-xhelp:///?Id=beckhoff-9857-4876-8432-cc4169af880e) | Addresses the control gears according to the random principle. The Bus Terminal’s internal addressing function is used here. |
| [FB\_DALIV2AddressingPhysicalSelection](ms-xhelp:///?Id=beckhoff-0137-4ed6-9dd7-14334d83c312) | Addresses the control gears according to the ‘physical selection’ addressing method. |
| [FB\_DALIV2AddressingRandomAddressing](ms-xhelp:///?Id=beckhoff-89e0-442c-bff4-24d9b263b726) | Addresses the control gears according to the random principle. |
| [FB\_DALIV2ChangeAddressList](ms-xhelp:///?Id=beckhoff-b7ff-4d47-a426-e08ecfd30549) | The short addresses of several control gears can be changed using this function block. |
| [FB\_DALIV2SwapShortAddress](ms-xhelp:///?Id=beckhoff-89c7-4eeb-97a5-510d13cd7241) | Swap the short addresses of two control gears. |
| [FB\_DALIV2SwapShortAddressList](ms-xhelp:///?Id=beckhoff-2df0-445f-9f7c-78cd65d9aa35) | Swaps the short addresses of several control gears. |
