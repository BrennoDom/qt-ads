# B.E.G.

## Function blocks

The company B.E.G. has defined its own special DALI commands. These commands extend beyond the possible DALI commands in accordance with the IEC 62386 standard. However, these commands can only be used for certain B.E.G devices. Please contact B.E.G for a more detailed description of the commands.

| Name | Description |
| --- | --- |
| [FB\_DALIV2xBEGLuxomat](ms-xhelp:///?Id=beckhoff-665d-4545-b347-f7bc80325e9d) | This function block evaluates the measured brightness and presence of the B.E.G. Luxomat DALI control unit. This function block can also be used to initialize the DALI control units. |
