# Theben HTS

## Function blocks

The company Theben has defined its own DALI commands. These commands extend beyond the possible DALI commands in accordance with the IEC 62386 standard. However, these commands can only be used for certain Theben devices. Please contact Theben for a more detailed description of the commands.

| Name | Description |
| --- | --- |
| [FB\_DALIV2xThebenPlanoSpot360](ms-xhelp:///?Id=beckhoff-cdce-4dc6-b74a-f65955b71a5c) | This function block evaluates the 3 measured brightness values and the presence of the ThebenHTS PlanoSpot DALI control unit. This function block can also be used to initialize the DALI control units. |
