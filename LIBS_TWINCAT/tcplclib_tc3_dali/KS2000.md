# KS2000

The KS2000 provides dialogs for commissioning and diagnosis of DALI devices on the KL6821.

The dialogs offer not only functions for addressing control gears and control devices, but also for writing and reading parameters. The search function automatically detects the device types and displays them in a tree structure.

![54045009](/tcplclib_tc3_dali/1033/Images/png/5977074059__Web.png)

All parameters of the KL6821 can also be set via the KS2000. This makes it possible, for example, to define the DALI commands that are sent when the digital inputs on the KL6821 are actuated.

![16643039](/tcplclib_tc3_dali/1033/Images/png/9007205231817995__Web.png)

Overview of the most important functions in the KS2000 for the KL6821:

* Find DALI devices
* Addressing the DALI devices, including adapting the short addresses
* Scene and group assignment of control gears
* Configuration of the control gears, including the parameters for the different device types
* Configuration of the control devices, including the parameters for the different sensor types
* Writing/reading of the memory banks
* Definition of DALI commands for DI1 and DI2 (separate for rising and falling edge)
* Definition of the DALI command for the K-bus watchdog (K-bus failure)
* Switching the internal DALI power supply unit on/off
* Activation/deactivation of the blocking of the process image for the PLC as soon as a DALI command is sent by actuating a digital input on the KL6821.
