# AddNull

This method adds the value zero to a property. Usually, a corresponding property was created beforehand with the method [AddKey()](ms-xhelp:///?Id=beckhoff-60aa-4221-b3cc-a3999a39c931).

## Syntax

```
METHOD AddNull
```

Sample call:

```
fbJson.AddKey('PropertyName');  
fbJson.AddNull();
```
