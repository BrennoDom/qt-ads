## Library
tf5430_tc3_xplanar

## Category
Motion Control

# Creating a PLC

:   A PLC must be created to control the mover, track or group, to create the geometry of an environment or to use Planar Feedback.

:   1. Select **PLC > Add New Item...**
![23182124](/tf5430_tc3_xplanar/1033/Images/png/9241971595__Web.png):   2. In the following dialog box, select **Standard PLC Project** and confirm with **OK**.![7312526](/tf5430_tc3_xplanar/1033/Images/png/9241974923__Web.png)
:   3. Add the libraries "[Tc3\_Physics](ms-xhelp:///?Id=beckhoff-5d84-47ec-beb4-21f6e21ea48c)" and "Tc3\_Mc3PlanarMotion" to the PLC project; see [Inserting libraries](ms-xhelp:///?Id=beckhoff-623a-435f-bb36-b392b0ea9fc5).

:   The PLC is created and you can issue commands to the corresponding objects as described in the following examples.
