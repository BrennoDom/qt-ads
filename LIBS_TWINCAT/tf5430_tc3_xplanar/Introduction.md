# Introduction

The TwinCAT 3 Planar Motion software package TF5430 is installed together with the software package TF5400.

## Target system

Windows 7/8/10 (only 64-bit)

## TwinCAT 3 Planar Motion Base

The TF5430 TwinCAT 3 Planar Motion software combines a wide range of functionalities for controlling XPlanar movers and enables efficient and intelligent implementation of individual XPlanar applications. TF5430 TwinCAT 3 Planar Motion is part of TF5890 TwinCAT 3 XPlanar. All associated function blocks are included in the library Tc3\_Mc3PlanarMotion, which is to be used in combination with the library [Tc3\_Physics](ms-xhelp:///?Id=beckhoff-5d84-47ec-beb4-21f6e21ea48c).

## Additional licensing requirements

TF5430 TwinCAT 3 Planar Motion requires the TC1250 license.
