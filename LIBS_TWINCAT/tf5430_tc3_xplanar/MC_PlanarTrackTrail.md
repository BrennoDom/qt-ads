# MC\_PlanarTrackTrail

A list of distinct tracks each starting at the ending vertex of its predecessor.

Do not call the main FB directly. Only use the available methods.

## 16050694 Methods

| Name | Description |
| --- | --- |
| [Clear](ms-xhelp:///?Id=beckhoff-c3f5-4773-8006-0f6d37a3b972) | Clears the TrackTrail. |
| [AddTrack](ms-xhelp:///?Id=beckhoff-a3d8-41cf-be7c-cb06221b2d4a) | Adds a track to the TrackTrail. The track should start at the end vertex of the currently last track. |
