# Introduction

The user of this library requires basic knowledge of the following:

* TwinCAT XAE
* PC and network knowledge
* Structure and properties of the Beckhoff Embedded PC and its Bus Terminal system
* Technology of DMX devices

The Tc2\_DMX library is usable on all hardware platforms that support TwinCAT 3.1 or higher.

Hardware documentation [EL6851](https://infosys.beckhoff.de/content/1033/el6851/index.html) in the Beckhoff information system.
