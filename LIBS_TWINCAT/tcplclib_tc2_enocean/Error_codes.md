# Error codes

## Library
tcplclib_tc2_enocean

## Category
Motion Control

# Error codes

| Value (hex) | Description |
| --- | --- |
| 0x0000 | No error. |
| 0x0001 | Checksum error. |
| 0x0002 | Watchdog monitoring. |
| 0x0003 | Buffer overflow (in the KL6023) |
| 0x0004 | No data received yet from sensor receive. |
