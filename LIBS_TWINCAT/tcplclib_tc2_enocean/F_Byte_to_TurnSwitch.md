# F\_Byte\_to\_TurnSwitch

![7416719](/tcplclib_tc2_enocean/1033/Images/gif/173399179__en-US__Web.gif)

This function converts a raw byte value to a Boolean array in the form of a data structure (see [STREnOceanTurnSwitch](ms-xhelp:///?Id=beckhoff-f1fc-41fb-bd39-05a81ecbe6da)).

## VAR\_INPUT

```
byData : BYTE;
```

**byData:** Raw data.
