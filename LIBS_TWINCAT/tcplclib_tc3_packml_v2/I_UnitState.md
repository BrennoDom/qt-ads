# I\_UnitState

This interface can be implemented in the unit function blocks of the application and makes available all methods of the Packaging State Model, which can then be filled as required with application code.

These methods are:

M\_Aborted

M\_Aborting

M\_Clearing

M\_Complete

M\_Completing

M\_Execute

M\_Held

M\_Holding

M\_Idle

M\_Resetting

M\_Starting

M\_StateComplete

M\_Stopped

M\_Stopping

M\_Suspended

M\_Suspending

M\_Undefined

M\_Unholding

M\_Unsuspending
