# Introduction

The TwinCAT 3 PLC library Tc3\_IPCDiag is used for the IPC diagnostics of Beckhoff IPCs.

The desired IPC diagnostic parameters are selected via an enumeration ([E\_IPCDiag\_ParameterKey](ms-xhelp:///?Id=beckhoff-a43e-4ae7-94cf-ccf5dddf0869)). Specify this enumeration as an input at the function blocks of the library.

Details of the possible diagnosis and the parameters can also be found in the documentation on the [Beckhoff Device Manager](https://infosys.beckhoff.com/content/1033/devicemanager/90071992810335755.html).
