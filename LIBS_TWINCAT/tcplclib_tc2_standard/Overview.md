# Overview

The Standard Library includes all IEC61131-3 POUs. The POUs can be classified in:

* Bistable Function Blocks
* Trigger Function Blocks
* Counter
* Timer
* Timer (LTIME)
* String Functions
* String Functions (WSTRING)
