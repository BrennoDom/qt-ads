# EIB\_PRIORITY

EIB telegram priority

```
TYPE EIB_PRIORITY :  
(  
 EIB_PRIORITY_LOW   := 1,  
 EIB_PRIORITY_HIGH  := 2,  
 EIB_PRIORITY_ALARM := 3  
)  
END_TYPE
```

**EIB\_PRIORITY\_LOW:** Priority low

**EIB\_PRIORITY\_HIGH:** Priority high

**EIB\_PRIORITY\_ALARM:** Priority alarm
