# Overview

The Tc2\_Math library contains extended mathematical functions for TwinCAT PLC.

## Functions

|  |  |
| --- | --- |
| [FLOOR](ms-xhelp:///?Id=beckhoff-3c7e-41cf-8de8-55ed4c8e2d35) | The FLOOR function determines a integral value from a floating point number that is a fraction smaller than or equal that number. |
| [FRAC](ms-xhelp:///?Id=beckhoff-7ed3-433a-a3c1-138050786179) | The FRAC function determines the decimal component of a floating point number. |
| [LMOD](ms-xhelp:///?Id=beckhoff-9b74-410d-b412-9059a6f98996) | The LMOD function carries out a modulo division and returns the signed residual. |
| [LTRUNC](ms-xhelp:///?Id=beckhoff-0c40-4823-b7f2-2c89a7014327) | The LTRUNC function determines the integral component of a floating point number. |
| [MODABS](ms-xhelp:///?Id=beckhoff-6324-45a2-a5e8-176d42e0e19c) | The MODABS function carries out a modulo division and determines the unsigned modulo value within the modulo range. |
| [MODTURNS](ms-xhelp:///?Id=beckhoff-e4b7-46cb-82e5-6cd982ca5e15) | The MODTURNS function carries out a modulo division and determines the signed integral component. |
| [F\_GetVersionTcMath](ms-xhelp:///?Id=beckhoff-8099-47a2-808f-facf7f96007f) | Returns version informations of the library |
