# ST\_FunctionGeneratorTB\_BkPlcMc (from V3.0.31)

This structure consolidates parameters for the time base of one or several function generators. A structure of this type is transferred to [MC\_FunctionGeneratorTB\_BkPlcMc](ms-xhelp:///?Id=beckhoff-fa42-43f7-96f2-1438c3255a9d)(), [MC\_FunctionGeneratorFD\_BkPlcMc](ms-xhelp:///?Id=beckhoff-41b5-46c5-8602-712cb9848321)() and [MC\_FunctionGeneratorSetFrq\_BkPlcMc](ms-xhelp:///?Id=beckhoff-b325-4204-a66a-f21961dd38a2)() function blocks.

```
TYPE ST_FunctionGeneratorTB_BkPlcMc :  
STRUCT  
     Frequency:        LREAL:=0.000001;  
     Freeze:           BOOL:=FALSE;  
  
     CycleCount:       DINT:=0;  
     CurrentTime:      LREAL:=0.0;  
     CurrentRatio:     LREAL:=0.0;  
END_STRUCT  
END_TYPE
```

**Frequency**: The operating frequency of the time base generated by an [MC\_FunctionGeneratorTB\_BkPlcMc](ms-xhelp:///?Id=beckhoff-fa42-43f7-96f2-1438c3255a9d)() function block in Hertz.

**Freeze**: If this variable is set to TRUE, a [MC\_FunctionGeneratorTB\_BkPlcMc](ms-xhelp:///?Id=beckhoff-fa42-43f7-96f2-1438c3255a9d)() function block will not evaluate the structure.

**CycleCount**: The number of fully generated signal sequences.

**CurrentTime**: The time elapsed since the currently created signal sequence.

**CurrentRatio**: The normalized progress since the start of the currently generated signal sequence.
