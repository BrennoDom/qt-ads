# Referencing

For incremental position measuring systems: Now at the latest, the axis should be referenced correctly and fully. Enter the index velocity, index direction, sync velocity, sync direction and the reference position under the **Encoder** tab. For more information see [MC\_Home\_BkPlcMc](ms-xhelp:///?Id=beckhoff-1b39-4911-8ef7-a4d128653cfd).
