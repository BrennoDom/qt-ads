# Global Constants

Constants for the Packaging Machine Tag Structures, they can’t be changed.

```
(* PMLa *)  
iMAX_CURRENT_MODE              : INT := 10  
iMAX_CUMMULATIVE_MODE          : INT := 10  
iMAX_CURRENT_STATE             : INT := 17  
iMAX_CUMMULATIVE_STATE         : INT := 17
```
