# ST\_KL6781inData22B

Process image of the inputs.

Linked to the terminals in the System Manager.

```
TYPE ST_KL6781inData22B :  
STRUCT  
  Status : WORD;  
  D      : ARRAY[0..21] OF BYTE;  
END_STRUCT  
END_TYPE
```

**Status:** Status word.

**D:** 22 bytes for the input data of the M-Bus.
