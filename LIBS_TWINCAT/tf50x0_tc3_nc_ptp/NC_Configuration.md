## Integrating and linking NC software packages or CNC software packages

In the dialog **EtherCAT drive(s) added**, which is described in the section [Automatically reading of drive amplifiers](ms-xhelp:///?Id=beckhoff-45ea-493f-bfa1-fe1ec35b8ad5), choose whether you wish to create an NC configuration or a CNC configuration in the MOTION subtree.
