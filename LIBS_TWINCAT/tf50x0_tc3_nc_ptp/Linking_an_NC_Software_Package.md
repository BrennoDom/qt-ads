# Linking an NC Software Package

## Opening the Settings dialog for an NC axis

:   1. Double-click the NC axis [1] for which you wish to open a dialog.
:   2. Select the **Settings** tab [1].

:   The **Settings** dialog opens.

![47528147](/tf50x0_tc3_nc_ptp/1033/Images/png/2598870667__Web.png)![25100141](/tf50x0_tc3_nc_ptp/1033/Images/png/2597686155__Web.png)

## Drop-down list box Axis Type

The drop-down list box **Axis Type** [1] assumes the value "Standard (Mapping via Encoder and Drive)" if the axis selected in the NC configuration is not linked with any device inside the IO configuration.

## Link To I/O… and Link To PLC… buttons

With the **Link To I/O…** button [2] you connect the interfaces of the NC software package with a servo drive inside the IO configuration.

With the **Link To PLC…** button [3] you connect the interfaces of the NC software package with an axis variable of a PLC project.
